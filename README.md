# Gusto Monorepo

This repository contains the source code for the Tayabli platform.

## Structure

- **apps/mobile**: Flutter application for Clients, Cooks, and Couriers.
- **apps/backend**: NestJS API Gateway and Services.

## Getting Started

### Mobile
```bash
cd apps/mobile
flutter pub get
flutter run
```

### Backend
```bash
cd apps/backend
npm install
npm run start:dev
```
